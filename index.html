What the skill does

Five steps an AI agent can perform, end-to-end, without touching Solidity.

1

Generate

Deterministic SVG generator. Same seed → same artwork. Inputs: shapes, palette, seed, size, background. Returns a well-formed SVG string.

2

Validate

Rejects <script> tags, javascript: URIs, and on*= event handlers. Enforces a 24 KB cap. Belt-and-suspenders with the on-chain check.

3

Deploy

Deploys OnchainSVG to Pharos. Foundry forge script, no Hardhat. Defaults to mainnet, switches to Atlantic testnet via the Makefile target.

4

Mint

Single NFT or batch (up to 50 per tx via mintBatch / mintBatchDistinct). Atomic — bad inputs revert the whole batch.

5

Read

Token metadata + SVG are returned inline from tokenURI as data:application/json;base64,…. No IPFS gateway, no off-chain dependency.

+

Secure by default

All writes are onlyOwner. SVG content sanitized both off-chain (validator) and on-chain (_enforceSafe). 24 KiB cap prevents out-of-gas mints.

Live SVG generator (same algorithm the agent uses)

Deterministic — pick a seed, you always get the same artwork. Re-runs as you type. No transactions, no wallet, no chain.

Ready — change a value to regenerate.

What an agent actually runs

Two paths: Foundry CLI for direct deploys, or the viem runtime when invoked by a Steward Agent.

Path A — Foundry CLI (direct, no Service Agent needed)

# 1. Clone
git clone https://github.com/Yemiight02/SVGM
cd SVGM

# 2. Build & test
forge build
forge test     # 29 tests pass: 24 unit + 1 property fuzz (256 runs) + 5 factory

# 3. Deploy to testnet
make deploy-collection-testnet NAME="Pharos Genesis" SYMBOL="PHG"
# → prints: collection : 0xNEW_ADDRESS

# 4. Mint single NFT
COLLECTION=0xNEW_ADDRESS RECIPIENT=0xYOUR_WALLET SVG_FILE=./art.svg \
TOKEN_NAME="Genesis #1" TOKEN_DESC="First onchain pixel" \
  make mint-testnet

# 4b. Mint batch of N (up to 50)
COLLECTION=0xNEW_ADDRESS RECIPIENT=0xYOUR_WALLET SVG_FILE=./art.svg \
COUNT=25 \
  make mint-batch-testnet

# 5. Read tokenURI inline
cast call 0xNEW_ADDRESS "tokenURI(uint256)(string)" 1 --rpc-url $PHAROS_RPC_URL
